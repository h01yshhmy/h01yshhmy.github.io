P2599 [ZJOI2009] 取石子游戏

不要把博弈做死了。灵活一点。

显然每个时刻的状态都是 \([1,n]\) 上的连续区间。

为了省事，以下简记 \(\langle i,j\rangle=a_i,a_{i+1},\cdots,a_j\)。

考虑设 \(L(i,j)\)：当前状态为：\((L(i,j),\langle i,j\rangle)\) 时，先手必败。

\(R(i,j)\) 同理：当前状态为：\((\langle i,j\rangle,R(i,j))\) 时，先手必败。

\(L,R\) 一定存在？如果存在是否唯一？

存在性：

反证。若存在一种情况使得 \(\langle i,j\rangle\) 左边放多少石子都先手必胜，则 \(L(i,j)\) 不一定存在。

在这种情况中，先手无论如何操作都会使当前状态变为必败态。

  • 若这一步拿了 \(\langle i,j\rangle\) 左边放的，当前状态为先手必败（先后手互换），那么 \(\langle i,j\rangle\) 左边剩下的一些石子就会成为 \(L(i,j)\)，与 \(L(i,j)\) 不存在矛盾。
  • 若这一步拿了 \(a_j\)，当前必败态，因为无论左边放多少都是先手必胜，后手继续在 \(\langle i,j\rangle\) 左侧拿石子，仍然为必败态送给先手。两个一步可达的状态均为必败态。若设初始先手A后手B，最初A必胜，而现在A必败，又是矛盾。

综上，不存在这种情况，也就证明 \(L\) 一定存在。\(R\) 同理。

唯一性：

反证。若 \(L(i,j)\) 不唯一，显然任意两种 \(L(i,j)\) 之间都一步可达，和存在性证明中第二种情况相同，矛盾。

\(L\) 一定唯一，\(R\) 同理。

显然求出 \(L,R\) 后答案可以随便判，考虑如何求出 \(L,R\)。

边界 \(L(i,i)=R(i,i)=a_i\)：退化为一般 Nim 游戏，两堆相等石子 Nim 和为 \(0\)。

剩下很多分讨。外层枚举区间长度，从底向上做。

设 \(L=L(i,j-1),R=R(i,j-1),x=a_j\)。

  • 若 \(x=R\)（Case 1）。

    • 有 \(L(i,j)=0\)。
    • 证明：右侧加入的 \(x\) 恰好满足先手必败，左边就不用加了。

  • 若 \(x<R\)：

    • 若 \(x<L\)，即 \(x<\min(L,R)\)（Case 2）。

      • 有 \(L(i,j)=x\)。

      • 证明：就是证明 \((x,\langle i,j-1\rangle,x)\) 为必败局面。

        两侧石子数相同，后手做对称操作一定最优（因为先手不傻）。

        直到当前局面形如 \((y,\langle i,j-1\rangle)\) 或者 \((\langle i,j-1\rangle,y)\)，显然这个局面轮到后手接盘。因为 \(0<y\le x<\min(L,R)\)，容易发现，后手必胜，先手必败，符合要求。

      • 注意必须满足 \(x>0\)。

    • 若 \(x\ge L\)，即 \(L\le x<R\)（Case 3）。

      • 有 \(L(i,j)=x+1\)。

      • 证明：就是证明 \((x+1,\langle i,j-1\rangle,x)\) 为必败局面。

        考虑两种情况：

        1. 若先手拿了 \(x+1\)，设还剩 \(y\) 个石子。

          若 \(y>L\)，后手让右侧剩下 \(y-1\) 个，递归证明。

          若 \(y=L\)，后手将右侧也拿完，和 \(L(i,j-1)\) 局面相同，必败局面。

          若 \(0<y<L\)，后手让右侧剩下 \(y\) 个，回到 Case 2，必败局面。

          若 \(y=0\)，后手无需操作，因为右边 \(L\le x<R\)，不符合 \(R(i,j-1)\)，先手必胜，必败局面。

        2. 若先手拿了 \(x\)，设还剩 \(y\) 个石子。

          若 \(y\ge L\)，后手让左侧剩下 \(y+1\) 个，递归证明。

          若 \(0<y<L\)，后手让左侧剩下 \(y\) 个，回到 Case 2，必败局面。

          若 \(y=0\)，后手让左侧剩下 \(L\) 个，符合 \(L(i,j-1)\)，先手必败，必败局面。

  • \(x>R\)：

    • \(x\le L\)，即 \(R<x\le L\)（Case 4）。

      • 有 \(L(i,j)=x-1\).
      • 证明：和 Case 3 思路完全相同，略。

    • \(x>L\)，即 \(x>\max(L,R)\)（Case 5）。

      • 有 \(L(i,j)=x\)。

      • 证明：就是证明 \((x,\langle i,j-1\rangle,x)\) 为必败状态。

        设先手让两侧中其中一个变成了 \(y\) 个石子。

        若 \(y>\max(L,R)\)，递归证明。

        若 \(0<y<\min(L,R)\)，后手让另一侧也剩 \(y\) 个，回到 Case 2，必败状态。

        若 \(y=0\)，后手让另一侧剩 \(L/R\) 个，必败状态。

        剩下 \(L\le x\le R\) 或 \(R\le x\le L\) 的情况。

        对于 Case 3，\(L\le x\le R-1\)，\((x+1,\langle i,j-1\rangle,x)\)。

        对于 Case 4，\(R+1\le x\le L\)，\((x-1,\langle i,j-1\rangle,x)\)。

        先手可以让两侧的 \(x\) 相差 \(1\) 来让当前状态适配 Case 3/4，必败状态。

        容易发现我们只需要额外解决 \(x=L\) 或 \(x=R\) 的情况。

        先手直接让 \(y=0\)，必败状态。

综上： \[ L(i,j)=\begin{cases}0&x=R\\x+1&L\le x<R\\x-1&R<x\le L\\x&\textrm{otherwise}\end{cases} \] \(R\) 同理，均为对称。

只需要判 \([L(2,n)=a_1]\) 即可。

时间复杂度 \(O(n^2)\)。
